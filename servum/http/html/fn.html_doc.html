pub fn html_doc<T, U, V>(title: T, lead: U, content: V) -> String where
    T: Display,
    U: Display,
    V: Display, 

Generate an HTML document containing title, lead and content.

This function is mainly used to generate HTML documents from HTTPStatus structs and send them to the consumer. This is especially useful for quickly generating 404, 500, etc... error pages.

Example

let doc = html_doc("Not Found", "404", "Page not found!");

assert!(doc.starts_with("<!DOCTYPE html>"));
assert!(doc.ends_with("</html>\n"));
assert!(doc.find("<h1>404</h1>").is_some());
